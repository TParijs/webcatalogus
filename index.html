Welcome to our games page

This is a website made for our project in period 3.

Project: Gaming


This project was a team effort involving four individuals working together to create a website. The website was comprised of various components, some of which were created collectively while others were individual efforts. The team worked together on the general pages, the corporate identity, and the layout of the website, using Bootstrap 5 for formatting.

Once the corporate identity and general pages were completed, the team members each focused on their individual parts. Each part consisted of three categories, with static pages and categories named by each team member. The pages were filled with data obtained from an API using JSON data and the Fetch method in JavaScript.

The successful completion of the website was a testament to the collaboration and teamwork of the four individuals involved. The combination of their efforts and the use of Bootstrap 5 for formatting resulted in a polished and functional website that met the team's goals and expectations. The use of JSON data and the Fetch method in JavaScript to populate some of the pages added an extra layer of complexity, but was successfully implemented to achieve the desired outcome.
